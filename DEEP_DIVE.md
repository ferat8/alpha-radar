# X402 Ecosystem — Deep Dive

## 🎯 Что такое X402?

**X402** — HTTP-native платёжный протокол от Coinbase для микроплатежей между AI агентами.

### Ключевые факты:
- Использует HTTP код **402 Payment Required**
- Расчёты в **USDC на Base** (~200ms settlement)
- **100M+ транзакций** между агентами (февраль 2026)
- **80+ проектов** в экосистеме
- **12,365+ сервисов** на Coinbase Bazaar
- Без API ключей, аккаунтов, подписок

## 🏗️ Как работает X402

```
┌─────────────┐    1. Request     ┌─────────────┐
│   Client    │ ────────────────► │   Server    │
│   (Agent)   │                   │  (Service)  │
└─────────────┘                   └─────────────┘
       │                                 │
       │         2. HTTP 402             │
       │ ◄───────────────────────────────┤
       │     Payment Required            │
       │     {price, payTo, network}     │
       │                                 │
       ▼                                 │
┌─────────────┐                          │
│   Sign      │                          │
│   Payment   │                          │
│   (USDC)    │                          │
└─────────────┘                          │
       │                                 │
       │    3. Request + Payment Header  │
       ├────────────────────────────────►│
       │                                 │
       │                          ┌──────▼──────┐
       │                          │ Facilitator │
       │                          │  (Verify +  │
       │                          │   Settle)   │
       │                          └──────┬──────┘
       │                                 │
       │         4. Response             │
       │ ◄───────────────────────────────┤
       │                                 │
```
